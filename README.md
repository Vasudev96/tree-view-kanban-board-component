# Kanban + Tree View (Drag & Drop)

A modern React + TypeScript project featuring:

- Sortable Kanban Board (dnd-kit)
- Recursive Tree View
- Drag & Drop functionality
- Editable nodes & cards
- Responsive UI

## Tech Stack

- React
- TypeScript
- dnd-kit
- CSS Modules
- Vite

## Features

- Drag cards between columns
- Drag tree nodes
- Inline editing
- Add / Delete nodes
- Responsive design

## Installation

```bash
npm install
npm run dev
```
